Constraints API
===============

.. currentmodule:: geotorch

These are the functions that form the basic interface of `GeoTorch`. They all provide a common
interface to the spaces implemented in `GeoTorch`. For a finer control over the underlying
implementation of each space, take a look at the different manifolds.
